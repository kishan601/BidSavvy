# **App Name**: BidSavvy

## Core Features:

- User Onboarding & Profile Management: Allow users to register as either a 'buyer' or 'seller' and manage their profiles, including relevant skills for sellers.
- Buyer Project Listing: Buyers can create detailed project descriptions, set a target budget, and specify deadlines for bids.
- Seller Bid Submission: Sellers can browse available projects and submit their competitive bids, including proposals and proposed timelines.
- Bid Review & Acceptance Dashboard: Buyers have a dashboard to review all submitted bids, compare offers, and select a winning seller.
- AI-Powered Project-Seller Matching: An AI tool that suggests relevant project listings to sellers based on their specified skills, profile, and past project categories.
- Basic Messaging System: Enable direct messaging between buyers and sellers for clarifications and communication during the bidding process.

## Style Guidelines:

- Primary color: A sophisticated, deep indigo (#5229A3) to convey trust and reliability.
- Background color: A very light, muted lavender-grey (#EEEDF3) providing a clean, professional canvas.
- Accent color: A vibrant, clear blue (#2C97EC) to highlight key actions, calls-to-action, and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, suitable for clear information display across all screen sizes.
- Use clean, outlined, modern icons to represent actions like 'submit bid', 'post project', 'accept offer', and user roles, maintaining a professional yet approachable feel.
- Employ a structured, card-based layout for project listings and bids, ensuring high readability and easy comparison of information, with clear division between sections.
- Subtle, smooth animations for form submissions, bid status updates, and interactive elements, providing clear feedback without distracting from the core functionality.
